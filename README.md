# FPGA VGA Demo

This project is a demo for using VGA on the Basys3 Digilent board.

# IHM
The first switch is the selector between Conway game and pattern.
The 16th and 15th switch are to enable column and line offset per frame.
The 14th switch is for selecting between manual frame rate and manual trigger using button up.
The second switch is for selecting conway game pattern init.

The reset button is the center button.
